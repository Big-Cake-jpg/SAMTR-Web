# 一号线

## 介绍

一号线是苏安市最早投入运营的地铁线路。

目前已完工的线路为一期工程，二期工程将在近期进行规划。

由苏安中铁轨道交通建设有限公司运营

|线路类型|地铁|
|-----------|-----------|
|所属公司|苏安中铁|
|全长|4.7km|
|状态|全线投入运营|
|站点总数|7 个|
